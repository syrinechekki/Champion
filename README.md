# Champion E-Book Website

A professional website for the Champion stretching and mobility e-book designed for students.

## Files Included
- index.html - Homepage
- about.html - About page
- team.html - Team page
- features.html - Features page
- styles.css - All styling
- script.js - JavaScript functionality
- logo.png - Logo image
- page1.png - Book cover image

## Deployment Options

### Option 1: Netlify (Easiest - Recommended)
1. Go to https://www.netlify.com
2. Sign up for free account
3. Drag and drop your entire Champion folder to Netlify
4. Your site will be live instantly with a free URL

### Option 2: GitHub Pages
1. Create a GitHub account at https://github.com
2. Create a new repository
3. Upload all your files
4. Go to Settings > Pages
5. Select main branch and save
6. Your site will be at: https://yourusername.github.io/repository-name

### Option 3: Vercel
1. Go to https://vercel.com
2. Sign up for free
3. Import your project folder
4. Deploy instantly
